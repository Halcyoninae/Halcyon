Halcyon

Copyright (C) Jack Meng (exoad) 2022

Halcyon is a robust & extensible music player & audio engine. It supports the most commonly used formats like:

  • MP3 (MPEG-Layer III)
  • WAV (WAVE)
  • OGG (Vorbis, Theora)
  • AIFF (Audio Interchange File Format, AIFC)
and many more...

It is written in C/C++ for the native backends, especially the audio frameworks and APIs.
Futhermore, the GUI is primarily handled by Java with certain rendering being deferred to native capabilities via JNI (Java Native Interface).
Plugin interfacing can be achieved through either Java or Lua (using LuaJ). All plugins are dynamically loaded at runtime.
